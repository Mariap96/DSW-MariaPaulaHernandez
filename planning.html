    ¿Qué es HTML5?

    Es un lenguaje markup usado para estructura y presentar el contenido para la web. Es la quinta revisión del estandar HTML. Con HTML5 tenemos otras posibilidades para explotar este recursos usando menos recursos. Con HTML5 entra en desuso el formato XHMTL, dado que su implementación ya no es necesaria.
    Le indica a los navegadores como mostrar una determinada pagina web, permite formatear el layout de cualquier pagina, asi como hacer algunos ajustes a su aspecto.
    La diferencia principal, sin embargo, es el nivel de sofisticación del código que podremos construir usando HTML5

    Etiquetas

    Metadatos del documento

    Etiqueta Descripción
    "head" Contiene una colección de metadatos acerca del documento
    "title" Define el titulo de la página que se va a observar en la barra del navegador
    "base" Define la URL base para las URL's relativas en la página
    "link" Usada para enlazar JavaScript y Css externos al documento
    "meta" Define los metadatos que no pueden ser definidos usando otro elemento HTML
    "style" Etiqueta usada para dar estilo con CSS
