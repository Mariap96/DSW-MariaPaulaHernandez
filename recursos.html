Editores de Código que te facilitaran usar HTML 5



1. Visual Studio Code


Es un editor de código de open source el cual es gratuito creado por Microsoft para los S.O. Windows, Linux y Mac.


Aqui te puedes dirigir a la pagina oficial de VSC



2. Sublime Text


Es un editor de código y de texto un poco basico pero permite la instalación de varios plugins que te pueden ayudar en tu proceso de desarrollo.Su uso es gratuito


Aqui te puedes dirigir a la pagina oficial de Sublime Text



3. Dreamweaver


Es un entorno de desarrollo integrado esta disponible para S.O. Mac y Windows. Su uso no es gratuito.


Aqui te puedes dirigir a la pagina oficial de Adobe Dreamweaver



4. Notepad++


Es un editor que cuenta con muchos plugins para extender las funciones. Es rápido y sumple. Su uso es gratuito y funciona en el S.O. Windows


Aqui te puedes dirigir a la pagina oficial de Notepad++




5. JetBrains Webstorm


Es un editor de código que cuenta con características como soporte para multitud de librerias Javascript como Node.js. Adicional a esto te permite editar el códifo y ver los cambios en tiempo real en el navegador. Sirve para los S.O. Mac, Linux y Windows.


Aqui te puedes dirigir a la pagina oficial de JetBrains Webstorm
